OutLook AnyFinder Ver0.9 - 사내 배포 패키지 README
============================================================

OutLook AnyFinder는 Microsoft Outlook 데스크톱 앱에 저장된 메일을 사용자 PC에 로컬 인덱싱하여 빠르게 검색하는 Windows 데스크톱 프로그램입니다.

이 패키지는 사내 직원이 Python 설치 없이 .exe로 실행할 수 있도록 구성된 배포용 패키지입니다.


1. 주요 기능
------------------------------------------------------------
- Outlook 메일 로컬 인덱싱
- 제목/본문/발신자/수신자/CC/첨부파일명 검색
- 기본 포함 검색
- '정확한 단어만' 옵션 지원
- '+' 멀티 검색 지원
  예: 바이오+견적서 = 바이오 AND 견적서
- 메일 주소 검색 및 인라인 자동완성
- 서명/본문 삽입 이미지를 첨부파일 목록에서 제외
- 받은편지함/보낸편지함/날짜/첨부파일 필터
- 검색 결과 카드형 UI 및 페이지 이동
- Outlook에서 원본 메일 열기
- 수동 동기화 및 자동 동기화
- 최초 동기화 이후 증분 동기화
- 동기화 진행률/스캔 진행률 표시
- 동기화 중지 지원
- 모던 다크 테마 / Mac OS, Linear 스타일 화이트 테마
- 모든 데이터는 사용자 PC에만 저장


2. 실행 파일 위치
------------------------------------------------------------
폴더 배포 방식인 경우:
  OutLookAnyFinder\OutLookAnyFinder.exe

단일 파일 배포 방식인 경우:
  OutLookAnyFinder.exe

권장 배포 방식은 폴더 배포 방식입니다.


3. 사용자 PC 요구사항
------------------------------------------------------------
- Windows 10 / 11
- Microsoft Outlook 데스크톱 앱 설치
- Outlook 계정 로그인 완료
- 디스크 여유 공간 500MB 이상 권장
- 메모리 4GB 이상 권장


4. 사용자 데이터 저장 위치
------------------------------------------------------------
인덱스 DB와 설정은 프로그램 폴더가 아니라 사용자별 로컬 경로에 저장됩니다.

  C:\Users\사용자명\.outlook_anyfinder\

주요 파일:
  anyfinder.db
  config.json

프로그램을 업데이트해도 이 데이터는 유지됩니다.


5. 최초 실행 안내
------------------------------------------------------------
최초 실행 시 동기화/인덱싱 대상 폴더와 범위를 선택합니다.

선택 가능:
  - 받은편지함
  - 보낸편지함
  - 임시보관함
  - 지운편지함
  - 하위 폴더 포함 여부
  - 3개월 / 6개월 / 1년 / 전체

메일이 많은 사용자는 처음에는 3개월 또는 6개월 범위를 권장합니다.


6. 사내 보안 안내
------------------------------------------------------------
코드 서명이 없는 내부 배포 파일은 Windows SmartScreen 또는 백신 경고가 표시될 수 있습니다.

경고가 표시되면 사내 배포 파일인지 확인 후:
  추가 정보 → 실행

사내 보안 정책에 따라 허용 목록 등록이 필요할 수 있습니다.
