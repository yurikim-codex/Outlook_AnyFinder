OutLook AnyFinder 사내 배포 담당자 체크리스트
============================================================

1. 빌드 환경
------------------------------------------------------------
필수:
  - Windows 10/11
  - Python 3.10 이상 권장
  - Microsoft Outlook 데스크톱 설치
  - Outlook 로그인 완료

설치:
  python -m pip install --upgrade pip
  python -m pip install -r requirements.txt
  python -m pip install pyinstaller


2. 권장 빌드 방식
------------------------------------------------------------
권장: onedir 방식

이유:
  - PyQt6/pywin32/Outlook COM 앱에서 안정적
  - 실행 속도 빠름
  - onefile보다 백신 오탐 가능성이 낮음
  - 문제 발생 시 원인 파악이 쉬움

빌드:
  build_release.bat

또는:
  powershell -ExecutionPolicy Bypass -File .\build_release.ps1

결과:
  release\OutLookAnyFinder_v0.9_YYYYMMDD_HHMM.zip


3. 단일 exe가 필요한 경우
------------------------------------------------------------
빌드:
  build_release.bat -OneFile

또는:
  python build_exe.py --onefile

주의:
  - 첫 실행이 느릴 수 있음
  - 백신/SmartScreen 오탐 가능성이 높아질 수 있음


4. 배포 패키지 구성 확인
------------------------------------------------------------
ZIP 내부에 아래 파일이 포함되어야 합니다.

  [ ] OutLookAnyFinder\OutLookAnyFinder.exe
  [ ] OutLookAnyFinder\_internal\
  [ ] README_사내배포.txt
  [ ] 실행_가이드.txt
  [ ] 문제해결_가이드.txt
  [ ] 배포담당자_체크리스트.txt

단일 exe 방식인 경우:
  [ ] OutLookAnyFinder.exe

------------------------------------------------------------
반드시 테스트 PC에서 확인합니다.

  [ ] 백그라운드 동기화 진행 표시 확인
  [ ] 동기화 중지 동작 확인
  [ ] 검색 결과 표시 확인
  [ ] 기본 포함 검색 확인
  [ ] 정확한 단어만 옵션 확인
  [ ] + 멀티 검색 확인
  [ ] 메일 주소 검색 확인
  [ ] 첨부파일 clip 표시 확인
  [ ] 받은편지함/보낸편지함 필터 확인
  [ ] 페이지 맨처음/마지막 이동 확인

  [ ] 자동 동기화 증분 동작 확인
  [ ] 데이터 초기화 후 폴더 카운트 초기화 확인
  [ ] 다크/화이트 테마 전환 확인


6. 사내 보안 안내
------------------------------------------------------------
코드 서명 전에는 SmartScreen 경고가 발생할 수 있습니다.
사내 배포 시 아래 중 하나를 권장합니다.

  - 사내 보안팀 허용 목록 등록
  - 배포 ZIP 해시 공유
  - 사내 공식 공유 드라이브/인트라넷에서만 배포
  - 필요 시 코드 서명 인증서 적용


7. 사용자 데이터 위치
------------------------------------------------------------
사용자별 데이터 저장 위치:

  C:\Users\사용자명\.outlook_anyfinder\

포함:
  - anyfinder.db
  - config.json

앱 업데이트 시 이 폴더는 기본적으로 건드리지 않습니다.


8. 업데이트 배포
------------------------------------------------------------
업데이트 방식:
  - 기존 프로그램 폴더 교체
  - 사용자 DB/설정은 유지

주의:
  - DB 스키마 변경 시 마이그레이션 필요
  - 대규모 업데이트 전 테스트 PC 검증 필수
  - release/, dist/, build/ 폴더는 GitHub에 업로드하지 않음

10. 릴리즈 후 사용자 안내 문구 예시
------------------------------------------------------------
1. ZIP 파일을 원하는 위치에 압축 해제합니다.
2. OutLookAnyFinder\OutLookAnyFinder.exe를 실행합니다.
3. Outlook이 로그인되어 있어야 합니다.
4. 최초 실행 시 동기화할 폴더와 범위를 선택합니다.
5. 문제가 있으면 문제해결_가이드.txt를 확인하세요.
