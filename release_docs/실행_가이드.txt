OutLook AnyFinder 실행 가이드
============================================================

1. 압축 해제
------------------------------------------------------------
전달받은 OutLookAnyFinder_v0.9_*.zip 파일을 원하는 위치에 압축 해제합니다.

예:
  C:\Tools\OutLookAnyFinder\

폴더 배포 방식에서는 exe만 따로 꺼내 실행하지 말고, 전체 폴더 구조를 유지하세요.


2. Outlook 준비
------------------------------------------------------------
프로그램 실행 전 Outlook 데스크톱 앱을 먼저 실행하고 아래를 확인합니다.

  - Outlook이 정상 실행되는지
  - 계정 로그인이 완료되어 있는지
  - 받은편지함/보낸편지함이 열리는지


3. 프로그램 실행
------------------------------------------------------------
폴더 배포 방식:
  OutLookAnyFinder\OutLookAnyFinder.exe 실행

단일 exe 방식:
  OutLookAnyFinder.exe 실행

Windows 보안 경고가 나오면 사내 배포 파일인지 확인 후:
  추가 정보 → 실행


4. 최초 실행 및 인덱싱
------------------------------------------------------------
최초 실행 시 동기화/인덱싱 대상 폴더와 범위를 선택합니다.

선택 가능:
  - 받은편지함
  - 보낸편지함
  - 임시보관함
  - 지운편지함
  - 하위 폴더 포함 여부
  - 인덱싱 범위: 3개월 / 6개월 / 1년 / 전체

인덱싱 중에는 진행률이 표시됩니다.

버튼:
  - 일시정지
  - 중단
  - 백그라운드

백그라운드를 선택하면 진행창은 닫히지만 동기화/인덱싱은 계속 진행됩니다.
진행 상황은 좌측 사이드바의 지운편지함 아래 진행 표시창에 표시됩니다.


5. 검색 사용
------------------------------------------------------------
검색창에 검색어를 입력하고 검색 버튼을 누릅니다.

기본 검색은 검색어가 포함된 메일을 찾습니다.

예:
  주관

검색 가능 예:
  주관
  주관기관
  사업주관

정확한 단어만 검색하고 싶으면 검색창 옆의 '정확한 단어만'을 체크합니다.


6. 멀티 검색
------------------------------------------------------------
'+'를 사용하면 여러 단어가 모두 포함된 메일을 검색합니다.

예:
  바이오+견적서

의미:
  바이오 AND 견적서


7. 메일 주소 검색
------------------------------------------------------------
메일 주소도 검색할 수 있습니다.

예:
  kim@company.com
  kim.cs@company.com
  kim+sales@company.com

검색 대상:
  - 보낸 사람 메일 주소
  - 받는 사람
  - CC

메일 주소는 검색창 내부에서 인라인 자동완성을 지원합니다.


8. 검색 결과 리스트
------------------------------------------------------------
검색 결과는 카드형 리스트로 표시됩니다.

각 카드에는 다음 정보가 표시됩니다.
  - 날짜
  - 폴더명
  - 제목
  - 보낸 사람/받는 사람
  - 첨부파일
  - 본문 일부

첨부파일은 clip 표시로 구분됩니다.

예:
  📎 견적서.xlsx
  📎 계약서.pdf
  📎 +2

서명 또는 본문 삽입 이미지로 판단되는 파일은 첨부파일 목록에서 제외됩니다.


9. 페이지 이동
------------------------------------------------------------
검색 결과가 많으면 하단에서 페이지를 이동할 수 있습니다.

버튼:
  - 맨처음
  - 이전
  - 다음
  - 마지막


10. 메일 상세 보기
------------------------------------------------------------
검색 결과 항목을 클릭하면 오른쪽에 메일 상세 내용이 표시됩니다.

지원 기능:
  - Outlook에서 열기

답장/전달 버튼은 제공하지 않습니다.


11. 수동 동기화
------------------------------------------------------------
좌측 사이드바에서 '동기화 시작'을 누릅니다.

동기화 대상:
  - 폴더
  - 하위 폴더 포함 여부
  - 동기화 범위

동기화 중:
  - 동기화 시작 버튼 비활성화
  - 동기화 중지 버튼 표시
  - 스캔/동기화 진행률 표시

최초 동기화 이후에는 이전 동기화 시각 이후 변경되거나 추가된 메일만 스캔합니다.


12. 자동 동기화
------------------------------------------------------------
설정에서 자동 동기화를 켤 수 있습니다.

주기:
  - 5분
  - 10분
  - 30분
  - 1시간
  - 수동만

자동 동기화가 켜져 있으면 다음 동기화까지 남은 시간이 표시됩니다.

예:
  자동동기화 09분58초

자동 동기화도 증분 방식으로 변경/추가된 메일만 확인합니다.


13. 설정
------------------------------------------------------------
좌측 하단 설정에서 다음을 변경할 수 있습니다.

  - 인덱싱 폴더 및 범위
  - 자동 동기화 여부 및 주기
  - 검색 결과 페이지 크기
  - UI 테마
  - 데이터 초기화

UI 테마:
  - 모던 다크 테마
  - 화이트 테마

화이트 테마는 Mac OS / Linear 스타일에 맞춰 구성되어 있습니다.


14. 데이터 저장 위치
------------------------------------------------------------
사용자 데이터는 아래 경로에 저장됩니다.

  C:\Users\사용자명\.outlook_anyfinder\

주요 파일:
  anyfinder.db
  config.json


15. 데이터 초기화
------------------------------------------------------------
설정 → 데이터 → 전체 데이터 삭제 및 초기화를 실행하면 인덱싱 데이터가 삭제됩니다.
초기화 후 다시 동기화를 진행해야 합니다.
