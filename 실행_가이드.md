# 📧 OutLook AnyFinder Ver0.9 — 실행 가이드

---

## 🖥 실행 환경 요구사항

| 항목 | 요구사항 |
|------|----------|
| **OS** | Windows 10 / 11 |
| **Python** | 3.10 이상 |
| **Outlook** | Microsoft Outlook 데스크톱 (설치 & 로그인 완료) |
| **메모리** | 4GB 이상 권장 |
| **디스크** | 500MB 이상 여유 공간 |

> ⚠ **이 프로그램은 Windows + Outlook 환경에서 동작합니다.**
> Outlook이 없는 환경(Linux/Mac)에서는 자동으로 Mock 모드(데모)로 실행됩니다.

---

## 🚀 실행 방법

### Step 1: 프로젝트 폴더로 이동

```cmd
cd outlook_search_project\outlook_anyfinder
```

### Step 2: 필수 패키지 설치

```cmd
pip install PyQt6 pywin32 beautifulsoup4
```

또는:

```cmd
pip install -r requirements.txt
```

### Step 3: 앱 실행

```cmd
python main.py
```

## 🔑 주요 단축키

| 단축키 | 동작 |
|--------|------|
| `Ctrl+K` | 검색 바로 포커스 (검색어 전체 선택) |
| `Ctrl+J` | 다음 검색 결과 선택 |
| `Ctrl+Shift+J` | 이전 검색 결과 선택 |
| `Esc` | 자동완성 닫기 |
| `Enter` | 검색 실행 / 자동완성 선택 |
| `↑↓` | 자동완성 목록 이동 |

## 🔍 검색 방법 예시

| 입력 | 동작 |
|------|------|
| `프로젝트 보고서` | 전체 검색 (제목+본문+발신자+첨부) |
| `from:김철수` | 김철수가 보낸 메일 |
| `to:이영희` | 이영희에게 보낸 메일 |
| `subject:견적서` | 제목에 "견적서" 포함 |
| `첨부:xlsx` | Excel 첨부파일 있는 메일 |
| `폴더:받은편지함` | 받은편지함만 |
| `날짜:2026-05` | 2026년 5월 메일 |
| `-반려` | "반려" 제외 |
| `"정확한 구문"` | 정확한 구문 검색 |
| `from:김철수 견적서 첨부:pdf` | 복합 조합 |

## ⚙ 설정 변경 방법

사이드바 하단 **⚙ 설정** 클릭:

| 탭 | 설정 항목 |
|----|----------|
| **📧 인덱싱** | 대상 폴더, 하위 폴더 포함, 인덱싱 범위 (3개월~전체) |
| **🔄 동기화** | 자동/수동 선택, 주기 (5분/10분/30분/1시간/수동만) |
| **🔍 검색** | 자동완성 최대 수, 페이지당 결과 수, 기본 정렬 |
| **💾 데이터** | DB 정보, 히스토리 초기화, FTS5 재구축, 전체 초기화 |

## ❓ 자주 묻는 질문

### Q: "Outlook 연결 실패" 메시지가 나옵니다
**A:** Outlook이 실행 중인지 확인하세요. Outlook을 먼저 열고 로그인한 후 앱을 재실행하세요.

### Q: 인덱싱이 너무 오래 걸립니다
**A:** [⏸ 일시정지] 후 나중에 재개하거나, [🔽 백그라운드]로 실행하세요. 이미 인덱싱된 메일은 바로 검색 가능합니다.

### Q: 새 메일이 검색이 안 됩니다
**A:** 사이드바 [지금 동기화] 버튼을 클릭하세요. 설정에서 자동 동기화를 켜면 주기적으로 업데이트됩니다.

### Q: Mock 모드란 무엇인가요?
**A:** Outlook 없이도 앱을 테스트할 수 있도록 15건의 샘플 메일 데이터로 실행되는 모드입니다. "나중에"를 선택하면 자동 진입합니다.

### Q: .exe로 빌드하고 싶습니다
**A:**
```cmd
pip install pyinstaller
python build_exe.py
```
→ `dist\OutLookAnyFinder.exe` 생성 (Python 없이 실행 가능)
